
\chapter*{Abstract}

\noindent This work investigates the interplay of lambda calculus, type theory, category theory, computation, and functional programming through the Curry-Howard-Lambek correspondence. It explores lambda calculus as a computational foundation, its expression in type-safe functional programming, and its categorical semantics, highlighting how these fields unify to enhance program design.

\vspace*{0.7cm}

\noindent \textit{Esta trabajo examina la interacción entre el cálculo lambda, la teoría de tipos, la teoría de categorías, la computación y la programación funcional a través de la correspondencia Curry-Howard-Lambek. Explora el cálculo lambda como base computacional, su expresión tipada y su semántica categórica, destacando cómo estos campos se unifican para mejorar el diseño de programas.}

\vspace*{1cm}

\noindent \textbf{Keywords}:
Lambda Calculus,
Type Theory,
Category Theory,
Computation,
Functional Programming,
Curry-Howard-Lambek Correspondence
