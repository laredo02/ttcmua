
\chapter*{Introduction}

This work explores the untyped and typed lambda calculus, intuitionistic logic, and the Curry--Howard Isomorphism. It then expands the correspondence adding categorical semantics into what is referred to as the Curry--Howard--Lambek Correspondence.

\begin{itemize}
\item The first chapter is an introduction to the \lcalc. It contains definitions and a few simple standard proofs. It is packed with examples that make it easy to read.

\item As for the second, it is the natural continuation of the first. Typing is introduced, as well as some normalization properties, differences with the untyped \lcalc \ are discussed.

\item The third chapter glances over intuitionistic logic and natural deduction. It introduces the Curry--Howard correspondence.

\item We continue with introductory topics to category theory, like functors and natural transformations. From there we cover the necessary universal constructions to build Closed Cartesian Categories.

\item The last chapter extends the Curry--Howard correspondence, establishing a relation between \lcalc \ and Closed Cartesian Categories.
\end{itemize}

This work is meant as an extension of the course on Theory of Programming Languages and a literature review to dive into category theory with research in mind. The approach we considered is to make the document accessible so other people can learn refer to it if they are interested.
